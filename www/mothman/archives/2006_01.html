objective-xpcom

so anyway, i had an idea to combine two things to create a more useful whole:

1) Objective-C, a very nice object oriented extension to C, instead of a C-like object-oriented language (C++), needs a framework to be fully functional (current frameworks are either utter crap (GNUStep) or proprietary (Cocoa/NextStep))
2) Mozilla NSPR/XPCOM/XUL - a pretty decent framework for building moderately cross platform applications. Doesn't do Objective-C, just C/C++(natively), JavaScript(XPConnect) and Java/Perl/Ruby/Python.

so the idea is to create an XPCOM binding for Objective-C, and from there build a decent Objective-C framework.

I'll start with a XULRunner build and go from there (cribbing from PlXPCOM along the way).
