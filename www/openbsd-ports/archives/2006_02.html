xulrunner examples

i put together a few xulrunner example projects that you can install like this:
# xulrunner --install-app application.xpi

these XPIs work with any 1.8.0.x version of Xul Runner on any of the platforms it runs on.

xulmine.xpi - Minesweeper clone (with sound)
exch-1.0.xpi - Exch, currency converter
chatzilla.xpi - ChatZilla, IRC client, the same one available as a Ff extension and as part of SeaMonkey.
